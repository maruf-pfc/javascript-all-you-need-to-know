# JS Logical Operators

## && (and) operator

```md
true && true; // true
true && false; // false
false && true; // false
false && false; // false
```

## || (or) operator

```md
true || true; // true
true || false; // true
false || true; // true
false || false; // false
```

## ! (not) operator

```md
!true; // false
!false; // true
```
